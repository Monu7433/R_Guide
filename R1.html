1. Basic R Operations

Essential commands for data management and package handling.

Import CSV

data <- read.csv("clinical_data.csv")

library(readxl) data <- read_excel("data.xlsx")

Purpose: Descriptive statistics overview.

Purpose: Convert variables to categorical (very common error).

data$group <- factor(data$group) data$gender <- factor(data$gender, levels=c("Male","Female"))

Purpose: Download new packages.

install.packages("dplyr")

Load Package

library(dplyr)

Export CSV

Purpose: Save results for publication.

write.csv(summary_stats, "results.csv", row.names=FALSE)

Save Model

Purpose: Save fitted models for later use.

save(lm_model, file="model.RData") saveRDS(cox_model, "cox_model.rds")

3. Data Manipulation using dplyr

Purpose: Clean, filter, and summarise clinical datasets before analysis.

Install and Load:

install.packages("dplyr") library(dplyr)

select()

Purpose: Select specific variables from dataset.

select(data, age, gender, bmi)

filter()

Purpose: Select observations based on condition.

filter(data, age > 50) filter(data, gender == "Male")

mutate()

Purpose: Create new variables.

mutate(data, bmi_group = ifelse(bmi < 25, "Normal", "Overweight"))

arrange()

Purpose: Sort data.

arrange(data, age) # Ascending arrange(data, desc(age)) # Descending

group_by() and summarise()

Purpose: Create summary tables.

data %>% group_by(treatment_group) %>% summarise(mean_age = mean(age))

Pipe Operator (%>%)

Purpose: Chain multiple operations.

data %>% filter(age > 50) %>% select(age, bmi) %>% arrange(desc(bmi))

5. Assumption Checking

Purpose: Ensure validity of statistical tests.

Normality Test (Shapiro-Wilk)

shapiro.test(data$outcome)

Interpretation:

  • p > 0.05: Normal distribution
  • p < 0.05: Use non-parametric test

Normality Visualization (Q-Q Plot)

qqnorm(data$variable) qqline(data$variable)

Points on diagonal line = normal distribution.

Equal Variance Test (Levene's)

library(car) leveneTest(outcome ~ group, data=data)

Interpretation:

  • p > 0.05: Equal variance
  • p < 0.05: Unequal variance (use Welch t-test)

6. Statistical Tests

Test When to Use R Command
Independent t-test Compare 2 independent groups t.test(outcome ~ group, data=data)
Paired t-test Same subjects, 2 time points t.test(before, after, paired=TRUE)
Chi-square 2 categorical variables chisq.test(table(var1, var2))
ANOVA Compare 3+ groups model <- aov(outcome ~ group, data=data)
summary(model)
Correlation Linear relationship cor.test(x, y, method="pearson")

Non-Parametric Alternatives

Parametric Non-Parametric R Command
Independent t-test Mann-Whitney U wilcox.test(outcome ~ group, data=data)
Paired t-test Wilcoxon Signed Rank wilcox.test(before, after, paired=TRUE)
ANOVA Kruskal-Wallis kruskal.test(outcome ~ group, data=data)

7. Effect Size

Purpose: Measure magnitude of difference.

library(effsize) cohen.d(outcome ~ group, data=data)

Interpretation (Cohen's d):

  • 0.2: Small effect
  • 0.5: Medium effect
  • 0.8: Large effect

8. Regression Analysis

Linear Regression

lm_model <- lm(outcome ~ predictor1 + predictor2, data=data) summary(lm_model)

R-squared: Percentage of variation explained.

Regression Diagnostics

Purpose: Validate regression assumptions.

# Diagnostic plots par(mfrow=c(2,2)) plot(lm_model) # Check multicollinearity library(car) vif(lm_model) # Normality of residuals shapiro.test(residuals(lm_model))

Check:

  • VIF < 5: No multicollinearity
  • Residuals normal: Shapiro p > 0.05
  • No pattern in residuals vs fitted plot

Logistic Regression

logit_model <- glm(disease ~ age + bmi, family=binomial, data=data) # Summary summary(logit_model) # Odds Ratios with CI exp(cbind(OR=coef(logit_model), confint(logit_model)))

Interpretation:

  • OR = 1: No effect
  • OR > 1: Increased odds
  • OR < 1: Decreased odds

9. Survival Analysis

Purpose: Analyze time-to-event outcomes.

Load survival package:

library(survival)

Kaplan-Meier

survfit(Surv(time, status) ~ group, data=data)

Cox Regression

cox_model <- coxph(Surv(time, status) ~ treatment + age, data=data) # Summary summary(cox_model) # Hazard Ratios exp(coef(cox_model)) # Confidence intervals exp(confint(cox_model))

Output: Hazard Ratio (HR)

  • HR > 1: Higher risk
  • HR < 1: Lower risk
  • HR = 1: No effect

Proportional Hazards Assumption

Purpose: Check Cox model validity (VERY IMPORTANT).

cox.zph(cox_model)

Interpretation:

  • p > 0.05: Assumption satisfied
  • p < 0.05: Violation - use time-dependent covariates

10. Baseline Characteristics Table (Table 1)

Purpose: Summarize patient characteristics by treatment group.

Method 1: dplyr

data %>% group_by(treatment_group) %>% summarise( N = n(), Age_Mean = mean(age), Age_SD = sd(age), BMI_Mean = mean(bmi), BMI_SD = sd(bmi), Male_N = sum(gender == "Male"), Female_N = sum(gender == "Female") )

Method 2: tableone package (PROFESSIONAL)

Standard in pharma and clinical trials.

library(tableone) CreateTableOne( vars = c("age", "bmi", "gender", "bp_systolic"), strata = "treatment_group", data = data, factorVars = c("gender") )

Used in:

  • Pharmaceutical research
  • Clinical trials
  • Journal publications

12. Sample Size Calculation

Purpose: Power analysis for study design (biostatisticians are hired for this).

Load pwr package:

library(pwr)

Two-sample t-test

pwr.t.test( d = 0.5, # Effect size (Cohen's d) power = 0.8, # Power (1-beta) sig.level = 0.05, # Alpha type = "two.sample" )

ANOVA

pwr.anova.test( k = 3, # Number of groups f = 0.25, # Effect size power = 0.8, sig.level = 0.05 )

Correlation

pwr.r.test( r = 0.3, # Expected correlation power = 0.8, sig.level = 0.05 )
